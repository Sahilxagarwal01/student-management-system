# 🎓 Student Management System

A full featured admin dashboard for managing student records, attendance, marks and fees.

## 🔗 Live Demo
[👉 Click Here](https://student-management-system-two-liart.vercel.app)

## ✨ Features
- 👨‍🎓 Add/Delete Students
- 📋 Attendance Tracking
- 📊 Marks & Auto Grades
- 💰 Fee Status Tracker
- 📈 Progress Bars
- 📧 Email Alerts
- 🖨️ Print Report
- 🌙 Dark/Light Mode
- 📱 Mobile Responsive

## 🛠️ Tech Stack
HTML | CSS | JavaScript | LocalStorage

## 🚀 How to Run
1. Clone the repository
2. Open index.html in browser
